## Getting Started

Install the module with: `npm install itpub-crawler`

```js
var crawler = require('itpub-crawler');
crawler.awesome(); // "awesome"
```

Install with cli command

```sh
$ npm install -g itpub-crawler
$ itpub --help
$ itpub --version
```
